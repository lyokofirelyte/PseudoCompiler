Pseudo Compiler for Linux & Mac

I have discovered a program called Mono that can be used to run the compiler on Mac & Linux machines. However, the auto-updater does not work on those machines, so you will have to manually download the latest version of the compiler when I release a new version.

Linux Instructions

Download the latest PseudoCompiler.exe and cd to that directory using a terminal.

sudo apt-get install mono-complete

mono PseudoCompiler.exe

Mac Instructions

Download and install Mono from the link above.

Download the latest PseudoCompiler.exe from the link above.

Open a terminal and cd to the directory that PseudoCompiler.exe is installed to

mono PseudoCompiler.exe
